# homebrew-pythinker

Homebrew tap for [Pythinker Code](https://github.com/mohamed-elkholy95/Pythinker-Code).

```sh
brew install mohamed-elkholy95/pythinker/pythinker-code
```

This tap is auto-updated by the
[homebrew-tap.yml](https://github.com/mohamed-elkholy95/Pythinker-Code/blob/main/.github/workflows/homebrew-tap.yml)
workflow on every `v*.*.*` tag. Do not hand-edit `Formula/*` — your
edits will be overwritten on the next release.
